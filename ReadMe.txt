
Для запуска программы требуется:

1. В папку "import_files" закинуть реестр.
2. Название файла реестра поменять на английский БЕЗ пробелов.
3. Выйти из реестра в excel, иначе программа не сможет его прочитать и не сработает!
4. Не открывать файлы, созданные программой до завершения ее работы.
5. Запустить программу и следовать дальнейшей инструкции.



Таблица реестра будет считать 17(Q) столбик - номер ОКВЕД, 18(R) столбик - номер ОКАТО.
